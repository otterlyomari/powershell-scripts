# PowerShell Scripts

## Introduction

Hello, and welcome to my collection of PowerShell scripts.

This repository exists primarily as a place to store utilities I've written while maintaining my own Windows systems. Many of these began as one-off solutions to problems I encountered, but eventually became useful enough to clean up, document, and share.

The goal isn't to build a huge framework or toolkit—it's simply to collect practical scripts that automate repetitive tasks, simplify troubleshooting, and make Windows administration a little less tedious.

Most scripts are written for PowerShell 7 and prioritize readability over cleverness.

Note: This is a work in progress, as such, treat the listed contents as a plan, not a confirmed list! This README will update when all scripts are complete.

## Repository Contents

The repository is organized into categories to make scripts easier to find.

### Windows

Utilities for Windows maintenance, cleanup, configuration, and repair.

Examples include:

- Software removal
- Registry cleanup
- Windows component management
- Service management

### Development

Scripts related to software development.

Examples include:

- Development environment setup
- SDK cleanup
- Build tool maintenance
- Package management

### Networking

Networking utilities and diagnostic scripts.

Examples include:

- DNS management
- Firewall configuration
- Network diagnostics
- VPN helpers

### Utilities

General-purpose scripts that don't fit elsewhere.

Examples include:

- File management
- System information
- Automation helpers
- Miscellaneous tools

## Philosophy

Every script in this repository follows a few basic principles whenever practical:

- Readable and well-commented
- Written with PowerShell 7 in mind
- Safe by default whenever possible
- Modular and easy to modify
- No unnecessary external dependencies

Some scripts perform destructive operations. Those scripts will clearly state their purpose and expected effects before execution.
